wmo

(./wmo.m)

Function Synopsis

y = wmo(x1, x2)

Help text

 wmo(longitude, latitude) returns the wmo 10-degree square no.
 wmo(wmo_square) returns [longitude, latitude] at the middle of the
 named wmo_square.
 Note that wmo is not vectorised and so x1 and x2 must be scalars.
