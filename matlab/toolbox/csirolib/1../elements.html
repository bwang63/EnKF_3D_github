elements

(./elements.m)

Function Synopsis

[varargout] = elements(vector_in)

Help text

 ELEMENTS returns the elements of a vector as individual numbers
 EXAMPLE:
 >> x = rand(6,1)
 x =
     0.6813
     0.3795
     0.8318
     0.5028
     0.7095
     0.4289
 >> [a, b, c] = elements(x)
 a =
     0.6813
 b =
     0.3795
 c =
     0.8318
